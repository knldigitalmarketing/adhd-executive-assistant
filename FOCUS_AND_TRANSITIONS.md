# Focus and Transitions

## Purpose

Help users start, continue, stop, and switch tasks.

Many ADHD users struggle not only with remembering tasks, but with disengaging from a task once focused.

## Core Concepts

### Focus Session

A planned block of time for one task, project, topic, or responsibility.

Examples:

- Work on motorcycle project
- Pay bills
- Exercise
- Clean kitchen
- Review email

Each focus session should include:

- Name
- Start time
- Planned end time
- Transition warning time
- Next task
- Priority
- Flexibility

## Transition Reminders

The system should remind the user before a focus session ends.

Examples:

- 15-minute warning
- 10-minute warning
- Stop now
- Switch to next task

## Timeline Behavior

The daily timeline should show both starts and stops.

Example:

8:00 AM  
Breakfast

9:00 AM  
Work on motorcycle fairing

10:20 AM  
Wrap up motorcycle work

10:30 AM  
Stop motorcycle work  
Next: Pay electric bill

## Product Rule

The assistant must treat stopping and switching tasks as first-class events, not secondary reminders.

## Goal

Reduce time blindness, hyperfocus problems, missed obligations, and difficulty transitioning.
