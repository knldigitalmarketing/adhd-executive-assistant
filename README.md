# ADHD Executive Assistant

First prototype skeleton based on the repository documentation.

## Scope

Included:

- App structure
- Data model skeleton from `DATA_MODEL.md`
- Onboarding flow from `INTERVIEW.md` and `INTERVIEW_FLOW.md`
- Daily briefing and responsibility dashboard from `DAILY_BRIEFING.md` and `RESPONSIBILITY_ENGINE.md`
- Today Timeline
- Focus sessions and transition reminders from `FOCUS_AND_TRANSITIONS.md`

Not included yet:

- Gmail integration
- Calendar integration
- AI scanning
- Shopping integrations
- Payment integrations

## Run

Use any local static server, or run:

```bash
node scripts/serve.mjs
```

Then open `http://127.0.0.1:4173`.

This prototype is dependency-free and uses static HTML, CSS, and JavaScript modules.
